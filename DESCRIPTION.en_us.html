Pixelspace is an original pixel display typeface drawn on a 5×7 grid with a built-in 1-pixel right-side bearing. It ships 93 glyphs — uppercase, lowercase, digits and common punctuation — in a single monospaced Regular weight, scaled so every "on" cell renders as a rounded 110-unit square inside a 125-unit pixel cell on an 875-unit em.

The design leans into the honest geometry of pixel type: each glyph-pixel is a softly rounded square (~10% corner radius) with a deliberate gap to its neighbors, giving the face a crisp, airy texture without any anti-aliasing or hinting tricks. It reads best at integer multiples of its native cell — 7×N CSS pixels (14px, 21px, 28px, 49px, 70px, 105px…) — for posters, headings, game UI, and any place you want text to feel like it came off a low-resolution display.
